#Device Usage Impact Visualizer

A Flask-based sustainability application that estimates energy consumption and carbon emissions based on device usage patterns.

## Features

* Device selection
* Energy consumption estimation
* Carbon emission calculation
* Interactive user interface
* Sustainability awareness tool

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## Supported Devices

* Laptop
* Smartphone
* Tablet
* Desktop Computer

## Inputs

Phone Usage (hours/day)
Laptop Usage (hours/day)
TV Usage (hours/day)

cd Device-Impact-Visualizer

python app.py
